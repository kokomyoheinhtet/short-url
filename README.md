# Getting Started

## Backend

### Run on local
1. go to `backend` directory
```shell
cd backend
```
2. Install dependencies
```shell
go mod tidy
```
3. run the server
```shell
go run main

```

## Frontend
### Run on local
1. go to `frontend` directory
```shell
cd frontend/shortern-url
```
2. Install dependencies
```shell
npm install
```
3. Run the server
```shell
npm run dev
```
